# Binary Search
---
## Outline
1. Finding a value in a sorted sequence.
	* Using Built-in methods (Collections and Arrays).  
	* Duplicates.
		* Finding first/last occurrence.
	* Lower bound and Upper bound.
2. Beyond Arrays
	* Monotonicity.
	* Simulating/Finding the answer using Binary Search.
		* Handling doubles.
